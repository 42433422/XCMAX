## 卡住"成为产品"的 3 件事

1. **底座没插电**：SSOT gate advisory、NeuroBus 保险默认关 → 无强制力，一切会回弹。
2. **schema 双头**：`ensure_*` 还能加列 + 生产 `FHD_SKIP_ALEMBIC=1` 绕过 alembic → 最底的砖在晃。
3. **运行时真相缺失**：没有"什么在跑 / 什么停了"的单一真相 → AI 看不见现实，才会空转。
