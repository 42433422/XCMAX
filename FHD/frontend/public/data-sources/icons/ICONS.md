Icons in this folder are custom, single-color silhouette designs created for functional differentiation only.
They do not reproduce or use any third-party brand colors or wordmarks.
Each SVG uses `currentColor` so it inherits the surrounding text color; the UI applies sizing and color via CSS.

If you add or modify icons, keep them:
- monochrome (use `fill="currentColor"`),
- single-path where possible,
- small (<= 1 KB ideally),
- 24x24 viewBox for consistency.

These marks are intentionally stylized to avoid trademark issues.
