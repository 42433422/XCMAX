# Legacy Pro Mode

This folder archives the original JARVIS/STARK professional-mode frontend.

It is intentionally kept separate from the new Mod-generated professional frontend so the old overlay can be restored, reused, or converted into a Mod later.

Compatibility shims remain in the old paths:

- `src/components/ProMode.vue`
- `src/composables/useProMode.ts`
- `src/stores/proMode.ts`
- `src/styles/pro-mode.css`
- `src/styles/pro-mode-enhanced.css`
- `src/styles/css/animations/pro-mode.css`
