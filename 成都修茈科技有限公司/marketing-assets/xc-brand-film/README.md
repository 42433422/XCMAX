# XC 18 秒品牌短片

## 成片规格

- 1920 × 1080
- 30 fps
- H.264 MP4
- 18 秒
- 原创电子氛围音轨
- 片尾官网与二维码：`https://xiu-ci.com`

## 分镜

1. `0–3s` 冰蓝光轨汇聚，XC logo 逐步点亮。
2. `3–7s` 穿过 logo 圆环，进入 AI 员工工作台。
3. `7–11s` 单据识别、客服回复、报表生成快切与三分屏。
4. `11–14s` 一个人被多个 AI 岗位节点环绕。
5. `14–18s` logo 回收定格，展示平台定位、官网与二维码。

## 重新构建

```bash
./build.sh
```

生成文件位于 `output/xc-brand-film-1080p.mp4`。
