## 应急升级路径

1. 静态 Skill 失败 → 进入动态阶段 → 超预算 → **人工介入**
2. 数据库/服务不可达 → 通知 admin
3. 安全事件（secrets 泄露、异常访问）→ 立即通知 admin + 执行 `security-secrets-guard` 应急流程
