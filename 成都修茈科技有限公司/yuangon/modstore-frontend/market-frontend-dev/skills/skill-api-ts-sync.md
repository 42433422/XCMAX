# ESkill：api.ts 接口同步（skill-api-ts-sync）

## 元信息

| 字段 | 值 |
|------|----|
| skill_id | `skill-api-ts-sync` |
| 所属员工 | `market-frontend-dev` |
| 业务域 | 前端 API 层与后端接口同步 |
| 版本 | 1.0.0 |

## 1. 静态阶段

**执行逻辑**：
```
读取后端蓝图 route 变更信息 → 在 api.ts 中定位对应函数
→ 更新 URL、请求体类型、响应类型
→ vue-tsc 编译检查 → 输出变更摘要
```

## 2. 动态触发条件

| 触发类型 | 规则 |
|----------|------|
| 执行报错 | 接口不存在；类型不兼容 |
| 结果不达标 | `ts_errors > 0` |

## 3. 固化

**验收标准**：`ts_errors == 0`，前端 API 调用与后端响应格式一致。
