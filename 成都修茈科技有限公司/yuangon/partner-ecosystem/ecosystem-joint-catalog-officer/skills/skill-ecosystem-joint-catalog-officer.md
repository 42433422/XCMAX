# 联合 Catalog 策展员技能

职责：O-B B2 生态联合 SKU · MODstore catalog 扩展 · 伙伴商品挂载与可见性策略。

## 执行步骤

1. 校验联合 Catalog 元数据、版本、授权与依赖。
2. 检查员工包/Mod 包可安装性及兼容边界。
3. 输出可发布、需整改、拒绝三态结论及证据。

## 输出契约

- summary：结论。
- evidence：真实文件、接口、记录或测试证据。
- risks：风险与不确定项。
- next_actions：下一步、负责人和是否需要人工确认。

没有真实证据时必须返回未验证，不得把计划、回显或合成事件计为成功。
