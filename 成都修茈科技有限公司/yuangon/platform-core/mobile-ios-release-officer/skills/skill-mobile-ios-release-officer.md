# iOS 发版员技能

职责：P-S iOS 渠道发布（规划中）：TestFlight / App Store 工程、notarize 协同与 release 门禁。

## 执行步骤

1. 检查 Xcode scheme、证书配置占位、版本号和归档条件。
2. 运行允许的静态检查与测试门禁。
3. 签名、上传和发布必须经人工确认，禁止读取私钥正文。

## 输出契约

- summary：结论。
- evidence：真实文件、接口、记录或测试证据。
- risks：风险与不确定项。
- next_actions：下一步、负责人和是否需要人工确认。

没有真实证据时必须返回未验证，不得把计划、回显或合成事件计为成功。
