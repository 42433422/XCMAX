# 任务提示词：更新 HTML 页面内容

## 使用场景

当需要修改营销页面文案、替换图片、调整 CTA 按钮或更新数据 JSON 时使用。

## 输入格式

```
任务：更新 HTML 内容
文件：<文件名，如 index.html>
位置描述：<用 CSS selector 或关键文字定位，如 ".hero-title" 或 "联系我们按钮">
当前内容：<原文或原路径>
目标内容：<新文案或新路径>
附加要求：<可选，如"保持 aria-label"、"图片需在 assets/ 下存在">
```

## 输出格式

```markdown
## 变更摘要

- 文件：`<filename>`
- 位置：`<selector/行号>`
- 原值：`<old>`
- 新值：`<new>`

## Diff

```diff
- <原内容>
+ <新内容>
```

## 校验结果

- W3C 错误数：0
- 断链图片：无
- JSON 格式（如适用）：✓
```

## 约束检查

在输出前，必须确认：
- [ ] 未修改 `app.py`、`nginx-*.conf` 或 `MODstore_deploy/**`
- [ ] 图片路径在 `assets/` 下存在（或已注明"待上传"）
- [ ] HTML 嵌套结构合法
- [ ] JSON 变更通过格式校验
